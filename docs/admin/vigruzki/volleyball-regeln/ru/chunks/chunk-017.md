ХАРАКТЕРИСТИКА ИГРЫ

Волейбол — это вид спорта, в который играют две команды
на площадке, разделённой сеткой. Существуют разные варианты
игры для особых условий, чтобы сделать её доступной для всех.

Цель игры — отправить мяч через сетку так, чтобы он коснулся
площадки на стороне соперника, и помешать сопернику сделать
то же самое. Команда имеет три касания для возврата мяча
(не считая контакта блока).

Розыгрыш начинается с подачи: подающий отправляет мяч через
сетку сопернику. Розыгрыш продолжается до тех пор, пока мяч
не коснётся площадки, не выйдет «в аут» или одна из команд не
сумеет правильно вернуть мяч.

В волейболе команда, выигравшая розыгрыш, получает очко
(система набора очков за каждый розыгрыш — Rally Point System).

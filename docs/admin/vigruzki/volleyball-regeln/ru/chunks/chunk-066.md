# Правила волейбола

19.4.2.2 Если действующий либеро становится неспособным играть, он/она
может быть заменён(а) регулярным заменяющим или сразу же на
площадке — вновь назначенным либеро. Вновь назначенный либеро
не имеет права играть ни на какой другой позиции до конца матча.

Если либеро не находится на площадке в момент объявления его
неспособным играть, новый либеро также может быть назначен.
Объявленный неспособным либеро не имеет права больше участвовать
в матче.

19.4.2.3 Тренер или, при его отсутствии, ассистент тренера либо игровой
капитан должны связаться со 2‑м судьёй и проинформировать его/её
о новом назначении.

19.4.2.4 Если вновь назначенный либеро становится неспособным играть или
объявляется таковым, допускаются последующие переназначения.

19.4.2.5 По просьбе тренера капитан команды может быть назначен новым
либeро.

19.4.2.6 В случае вновь назначенного либеро номер этого игрока должен быть
занесён в графу «Примечания» протокола матча и в контрольный
лист либеро (или электронный протокол, если он используется).

19.4.3 Команда с двумя либеро

19.4.3.1 Если в протокол команды внесены два либеро, но один из них не
может играть, команда может продолжать матч с одним либеро.

Переназначение допускается только в том случае, если оставшийся
либеро становится неспособным играть.

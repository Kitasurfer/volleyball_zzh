SPIELFORMAT
6.1.1 Punkt
Eine Mannschaft erzielt einen Punkt:
6.1.1.1 indem er den Ball erfolgreich auf dem gegnerischen Spielfeld landet;
6.1.1.2 wenn die gegnerische Mannschaft einen Fehler begeht;
6.1.1.3 wenn die gegnerische Mannschaft einen Elfmeter erhält.
6.1.2 Störung
Eine Mannschaft begeht einen Fehler, wenn sie eine regelwidrige Spielhandlung 
vornimmt (oder auf andere gegen die Regeln verstößt). Die Schiedsrichter beurteilen 
die Fehler und legen die Konsequenzen gemäß den Regeln fest:
6.1.2.1 Werden zwei oder mehr Fehler nacheinander begangen, wird nur der erste Fehler

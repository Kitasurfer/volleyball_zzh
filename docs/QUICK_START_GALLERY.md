## 🐛 Troubleshooting

### Изображения не грузятся?
```typescript
// Проверьте src в изображениях
console.log(images[0].src)
```

### Lazy loading не работает?
```typescript
// Убедитесь что loading="lazy" есть
<img loading="lazy" src={...} />
```

### Swipe не работает на мобильном?
```typescript
// Проверьте touch handlers
onTouchStart={onTouchStart}
onTouchMove={onTouchMove}  
onTouchEnd={onTouchEnd}
```

### Zoom странно себя ведет?
```typescript
// Убедитесь что zoom state обнуляется
useEffect(() => {
  setZoom(1);
}, [currentIndex]);
```

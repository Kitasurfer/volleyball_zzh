# Правила волейбола

**Дриблинг (dribbling)**  
Дриблинг означает, что мяч отскакивает от пола (обычно в качестве подготовки к броску и подаче). К другим подготовительным действиям могут относиться, среди прочего, перекладывание мяча из руки в руку.

**Подающие мячи (ball retrievers)**  
Лица, обязанностью которых является поддерживать непрерывность игры, перекатывая мяч подающему между розыгрышами.

**Дежурные по уборке пола (floor wipers)**  
Сотрудники, задача которых — после каждого розыгрыша при необходимости очищать и просушивать площадку при помощи небольших полотенец.

**Система «ралли‑поинт» (rally point)**  
Система счёта, при которой очко начисляется команде, выигравшей розыгрыш.

**Интервал**  
Время между партиями. Смена сторон в пятой (решающей) партии не считается интервалом.

**Переименование (re‑designation) либеро**  
Процедура, при которой либеро, не способный продолжать игру или объявленный командой «неспособным играть», заменяется в роли либеро другим игроком (кроме регулярного заменяющего), который в момент такого назначения не находится на площадке.

**Замена (libero replacement)**  
Действие, при котором регулярный игрок покидает площадку, и один из двух либеро (если их два) занимает его место. Это может включать и обмен «либеро на либеро». Затем данный регулярный игрок может заменить одного из двух либеро. Между такими заменами должен быть завершён как минимум один розыгрыш …

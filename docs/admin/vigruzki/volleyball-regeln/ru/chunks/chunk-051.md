# Правила волейбола

13.2 ОГРАНИЧЕНИЯ АТАКУЮЩЕГО УДАРА

13.2.1 Игрок передней линии может выполнять атакующий удар на любой
высоте при условии, что контакт с мячом произошёл в пределах
собственного игрового пространства (за исключением правил
13.2.4 и 13.3.6).

13.2.2 Игрок задней линии может выполнять атакующий удар на любой
высоте из‑за предела передней зоны:

13.2.2.1 стопа (стопы) игрока при отталкивании не должна касаться или
пересекать линию атаки;

13.2.2.2 после удара игрок может приземлиться в передней зоне.

13.2.3 Игрок задней линии также может выполнять атакующий удар из
передней зоны, если в момент контакта часть мяча находится ниже
верхнего края сетки.

13.2.4 Ни одному игроку не разрешается выполнять атакующий удар по
подаче соперника, когда мяч находится в передней зоне и целиком
выше верхнего края сетки.

13.3 ОШИБКИ ПРИ АТАКУЮЩЕМ УДАРЕ

13.3.1 Игрок посылает мяч на сторону соперника за пределы площадки.

13.3.2 Игрок отправляет мяч «в аут».

13.3.3 Игрок задней линии выполняет атакующий удар из передней зоны,
когда мяч в момент удара находится полностью выше верхнего края
сетки.

13.3.4 Игрок выполняет атакующий удар по подаче соперника, когда мяч
находится в передней зоне и полностью выше верхнего края сетки.

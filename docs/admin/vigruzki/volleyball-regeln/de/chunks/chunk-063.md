19.3.1.1 Der Libero darf jeden Spieler auf einer Position in der hinteren Reihe ersetzen.

19.3.2 Libero-Ersatzspieler
19.3.2.1 Libero-Auswechslungen werden nicht als Auswechslungen gezählt.
Sie sind unbegrenzt, aber es muss ein abgeschlossener Ballwechsel zwischen 
zwei Libero-Ersatzspielern stattfinden (es sei denn, eine Strafe führt dazu, dass 
das Team rotiert und der Libero auf die vierte Position rückt, oder der amtierende 
Libero wird spielunfähig, wodurch der Ballwechsel unvollständig wird).

## 🎯 Интерактивность

### Действия пользователя:

1. **По умолчанию**: Показывается карта зала (первый таб активен)
   - Таб "Halle" подсвечен синим цветом
   - Под картой отображается адрес зала

2. **Клик на таб "Beach"**:
   - Таб "Beach" становится активным (синий)
   - Таб "Halle" деактивируется (серый)
   - Карта плавно меняется на пляжную локацию
   - Информация под картой обновляется

3. **Hover на неактивный таб**:
   - Цвет меняется с серого на темно-серый
   - Курсор становится pointer
   - Подсказывает о возможности клика

- **Неактивный**: `#525252` (neutral-600) - темно-серый

- **Lazy loading** для iframe (loading="lazy")

- **CSS transitions** вместо JavaScript анимаций

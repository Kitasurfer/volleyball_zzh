14.1 BLOCKIEREN
14.1.1 Blocken ist die Aktion von Spielern in der Nähe des Netzes, um den vom Gegner 
kommenden Ball abzufangen, indem sie höher als die Oberkante des Netzes 
greifen, unabhängig von der Höhe des Ballkontakts. Nur Spieler der vorderen Reihe 
dürfen einen Block ausführen, aber im Moment des Ballkontakts muss ein Teil des 
Körpers höher als die Netzoberkante sein.

Ein Block ist abgeschlossen, wenn der Ball von einem Blocker berührt wird.
14.1.4 Kollektiver Block
Ein kollektiver Block wird von zwei oder drei Spielern nahe beieinander ausgeführt und ist 
abgeschlossen, wenn einer von ihnen den Ball berührt.

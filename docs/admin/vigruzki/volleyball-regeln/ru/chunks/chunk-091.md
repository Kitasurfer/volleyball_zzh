# Правила волейбола

СХЕМА 11: ОФИЦИАЛЬНЫЕ РУЧНЫЕ ЖЕСТЫ СУДЕЙ (ЧАСТЬ 4)

24. **Касание мяча**  
Провести ладонью одной руки по пальцам другой руки, удерживаемой вертикально.

25. **Предупреждение / штраф за задержку**  
Накрыть запястье жёлтой карточкой (предупреждение) и красной карточкой (штраф).

СХЕМА 12: ОФИЦИАЛЬНЫЕ ФЛАГОВЫЕ СИГНАЛЫ ЛИНЕЙНЫХ СУДЕЙ

Обозначение: **L** – линейный судья.

1. **Мяч «в поле»**  
Опустить флажок вниз, указывая на площадку.

2. **Мяч «аут»**  
Поднять флажок вертикально.

3. **Касание мяча**  
Поднять флажок и свободной рукой коснуться его вершины.

4. **Ошибка при пересечении плоскости сетки / мяч касается внешнего предмета / ошибка ногой любого игрока при подаче**  
Помахать флажком над головой и затем указать им на антенну или соответствующую линию.

5. **Невозможно вынести решение**  
Поднять и скрестить руки и кисти перед грудью.

ЧАСТЬ 3 – ОПРЕДЕЛЕНИЯ

В этой части даются определения следующих терминов: зоны, области, нижнее пространство, пространство перехода, внешнее пространство, зона замен, исключения FIVB, стандарты FIVB, ошибка, дриблинг.

**Зона соревнований / контроля**  
Зона соревнований / контроля — это коридор вокруг площадки и свободной зоны, включающий все пространства вплоть до внешних …

# ❌ ПРОБЛЕМА

## VLW сайт блокирует Supabase Edge Functions

League-results падает с 500 потому что:
- VLW сайт блокирует запросы от Supabase Edge Runtime
- Локально парсинг работает отлично
- В Supabase - 500 ошибка

## Решения:

### 1. Использовать прокси сервер
Развернуть простой прокси который будет делать запросы к VLW от вашего имени

### 2. Кешировать данные в Supabase
Создать таблицу в Supabase и обновлять её раз в день через cron job

### 3. Использовать другой источник данных
Найти API или RSS feed от VLW

### 4. Запускать локально
Держать сервер у себя который будет обновлять данные

---

## Что работает:

- ✅ **Chatbot** - использует Cerebras `qwen-3-235b-a22b-thinking-2507`
- ✅ **Ingest-content** - генерирует embeddings математически
- ✅ **Парсинг VLW** - работает локально идеально
- ❌ **League-results в Supabase** - блокируется VLW

---

## Рекомендация:

Создать cron job который будет запускаться локально раз в день и обновлять таблицу в Supabase с данными лиг.

Или использовать Netlify Functions вместо Supabase Edge Functions - они могут не блокироваться.

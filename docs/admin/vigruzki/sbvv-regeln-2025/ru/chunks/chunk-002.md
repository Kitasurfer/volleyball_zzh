# Изменения правил 2025 – SBVV 250724 193703 > 1. Позиции (Правило 7.4)

Регламент расположения игроков на площадке изменён так, что **только** подающая команда освобождается от соблюдения предписанных позиций ротации в момент подачи.  
**Правило 7.4**: В момент удара подающего по мячу каждая команда должна находиться на своей половине площадки (кроме подающего). Игроки принимающей команды обязаны располагаться согласно порядку ротации в момент подачи.  
Однако игроки подающей команды могут занимать любые позиции во время подачи.  
Порядок подачи остаётся обязательным: правильный подающий должен выходить строго по ротации. Следовательно, никаких изменений в определении ошибок ротации (не тот подающий) или позиционных ошибок принимающей команды нет.
